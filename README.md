# Partaches

Cette application a pour but de faciliter le partage des tâches dans un foyer

## Comment est prévu l'avancé du projet 

On développe sur une branche. 
On fait une PR pour main. Ceci va jouer une CI pour s'assurer de quelques basiques.

Puis, une fois une quantité suffisament intéressante d'avancé faite, main doit être merge à la main sur livraison, ce qui déclenche la CD et assure la livraison du code
